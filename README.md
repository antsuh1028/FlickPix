# FlickPix
Mobile application that suggests movies to watch based on past movie history

## React Native Setup (Expo)

This repository is set up as a basic React Native app using Expo + JavaScript.

### Prerequisites
- Node.js 20+
- npm 10+
- Expo Go app on your phone (optional for mobile testing)

### Run
```bash
npm install
npm start
```

### Open app
```bash
npm run android
npm run web
```

If you use `npm start`, scan the QR code from the terminal with Expo Go.
